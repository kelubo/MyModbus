## 📦 项目概述

Modbus RTU Manager 是一个功能完整的工业设备管理系统，支持 Modbus RTU 和 TCP 协议，提供设备管理、数据采集、实时监控、告警管理等全套功能。

## ✅ 已完成功能清单

### 核心功能（12项）

1. ✅ **设备管理** - 完整的 CRUD 操作，支持 RTU 和 TCP
2. ✅ **数据采集** - 自动采集，实时推送，数据存储
3. ✅ **实时监控** - WebSocket 推送，图表展示
4. ✅ **设备拓扑图** - SVG 可视化，交互操作
5. ✅ **告警管理** - 规则配置，多级别告警
6. ✅ **告警通知** - 邮件/短信/企业微信/钉钉
7. ✅ **系统设置** - Logo/主题色/时间格式
8. ✅ **系统监控** - CPU/内存/磁盘/网络
9. ✅ **集群监控** - 节点状态，任务分配
10. ✅ **备份还原** - 数据库备份，一键还原
11. ✅ **远程配置** - 寄存器写入，IP 修改
12. ✅ **多数据库** - SQLite/MySQL/PostgreSQL
